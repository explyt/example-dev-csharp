# AuthService Tests

Этот проект содержит unit-тесты для сервиса аутентификации `AuthService`.

## Покрытые сценарии

### Успешная аутентификация
- Тестирует аутентификацию для всех захардкоженных пользователей:
  - `jimmy.solid` / `secret`
  - `danny.solid` / `secret` 
  - `admin` / `admin`
- Проверяет корректность возвращаемых данных (токен, роли, аватар, время истечения)

### Неуспешная аутентификация
- Неверный пароль для существующих пользователей
- Несуществующие логины
- Null значения для логина и пароля

### Дополнительные тесты
- Проверка метода `AgentFromLogin`
- Валидация формата JWT токена
- Проверка claims в токене

## Запуск тестов

```bash
cd AuthService.Test
dotnet test
```

## Зависимости

- xUnit - фреймворк для тестирования
- FluentAssertions - библиотека для читаемых assertions
- Microsoft.Extensions.Options - для работы с настройками
