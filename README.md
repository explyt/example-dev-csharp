## Требования 


- .NET 8.0


## Запуск приложения


1. Выполните `dotnet build`
2. Перейдите в *MicroservicesOrchestrator* и выполните `dotnet run`
 

## После запуска 

Приложение будет доступно по адресу **http://localhost:5081**

**Учетные данные:**
- Username: `admin` 
- Password: `admin` 


## Тестирование

Проект покрыт тестами, для их запуска необходимо выполнить
```bash
dotnet test
```
